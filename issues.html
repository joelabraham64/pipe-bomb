Paralign shutting off

This can be fixed in a few ways.
1. Turn off/on the device
2. Switch the batteries
3. thats about it

No Plate

1. Make your own plate
Just as long as it is in a stable place and is still parallel with the link roll you will be fine. 
You can check this by looking at the results tab on the Paralign Editor software

Bad link roll/link roll was moved

This can be fixed in a few ways.
1. Create 2 pdas and merge them together later
2. Create a master roll

Session gets deleted

This can be fixed in a few ways.
1. start again or merge the pdas if possible

Won't connect to laptop

This can be fixed in a few ways.
1. Turn off/on bluetooth
2. make sure you are using the right COM ports and baud rate 11560
3. Use the antane/dongle
4. have a wired connection between the device and laptop
5. stop and start pc conecction

Drop the paralign

It will say if it has had a 30 and 50 G shock and display on the lcd screen so don't be worried just yet.
1. Check for any damages to the paralign especially the feet
2. Perform a cone angle check
3. Send back to Germany? They will do the calibration thingy to offset it correctly
